# 课程导学

## 产品需求

+ 技术选型
+ 项目架构
+ 项目规范
+ 封装公共机制
+ 提高开发效率

## 课程概述

+ React全家桶
  + React基础知识、生命周期
  + Router4.0语法讲解
  + Redux集成开发
+ AntD UI组件
  + 最实用基础组件
  + AntD栅格系统
  + ETable组件封装
  + BaseForm组件封装
  + 表格内嵌单选、复选封装
+ 公共机制封装
  + Axios请求插件封装
  + API封装
  + 错误拦截
  + 权限、菜单封装
  + 金额、日期、手机号封装......
  + Loading、分页、Mock......
