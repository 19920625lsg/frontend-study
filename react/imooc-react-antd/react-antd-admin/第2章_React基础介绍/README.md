# React基础

## 关于React

+ Facebook 开源的一个JavaScript库
+ React 结合生态库构成一个 MV *框架
+ React 特点
  + Declarative(声明式编码)
  + Component-Based(组件化编码)
  + 高效一高效的 DOM Diff算法,最小化页面重绘
